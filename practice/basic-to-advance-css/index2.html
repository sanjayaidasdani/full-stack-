CSS Size Units in Depth: Unlocking px, %, vw, vh, em, rem

Introduction

CSS size units control dimensions like width, height, font-size, padding. They fall into absolute (fixed, e.g., px) and relative (scalable, e.g., %, vw). Resize the browser to see differences!

1. px (Pixels) - Absolute Fixed Size

What: 1/96th of 1in; fixed regardless of screen/font.
Use: Precise control (icons, borders). Not responsive.

200px x 100px (Fixed - resize window, stays same)

2. % (Percentage) - Relative to Parent

What: % of parent element's width/height/font-size.
Use: Flexible layouts (containers filling parent).

50% of parent (400px wide)

3. vw (Viewport Width) & 4. vh (Viewport Height) - Relative to Viewport

vw: 1% of viewport width. vh: 1% of viewport height.
Use: Fullscreen/responsive designs.

50vw x 20vh (Resize viewport to see!)
30vw x 50vh

5. em - Relative to Parent Font-Size

What: 1em = parent element's font-size. Cascades (child em relative to child).
Use: Compound components. Can compound unexpectedly.

Parent: 20px, this: 1.5em (30px)

20em width x 10em height (relative to own font-size)

6. rem - Relative to Root Font-Size

What: 1rem = font-size (usually 16px). Consistent across page.
Use: Scalable typography, accessibility (user font resize).

15rem x 8rem (scales with root)
